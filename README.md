# GitGoing

A simple todo/task tracker I built to practice Git workflows.

## What it is

It's a basic web app where you can add, check off, and delete tasks. Nothing fancy -- just vanilla HTML, CSS, and JavaScript. Tasks are saved in localStorage so they stick around between sessions.

## How to run it

Just open `index.html` in your browser. That's it. No build tools, no dependencies, no server needed.

## Why I made this

I wanted a small project to practice branching, committing, pushing, pull requests -- the whole Git flow. The app itself is intentionally simple so I could focus on the workflow side of things.
